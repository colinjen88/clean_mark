# Gemini 浮水印去除工具 (專業強化版 - Phase 2)

本工具是專為 Google Gemini 生成的圖像與影片設計的進階去除方案，採用 Gradio 框架重構，提供更穩定的操作體驗與專業級的 AI 修復效果。

## 🌟 核心升級
1. **框架重構**：全面遷移至 **Gradio 6.x**，解決了舊版 Streamlit 畫布組件不兼容的問題。
2. **數據傳輸優化**：改採實體路徑傳輸模式，徹底杜絕瀏覽器圖片解碼錯誤 (`InvalidStateError`)。
3. **API 深度掛鉤**：精準適配 **MoviePy 2.x** 新版影片處理引擎 (`image_transform`)。
4. **RGBA 智慧遮罩**：原生支持帶有透明通道的塗抹遮罩，自動識別繪圖層。

## 🛠️ 功能模組說明

### 1. 🖼️ 智能圖片修復
- **操作方式**：上傳圖片後，直接使用紅色畫筆塗抹浮水印區域。
- **技術細節**：使用 OpenCV Telea 高速修復算法，自動計算邊緣權重進行像素填補。

### 2. 📦 批量處理工廠
- **適用場景**：當您有大量相同位置浮水印的圖片（如 Gemini 批量生成的作品集）時使用。
- **操作方式**：
    1. 上傳一張參考圖並塗抹遮罩。
    2. 批次上傳多張待處理圖片。
    3. 系統將自動套用遮罩並產出 `.zip` 壓縮檔。

### 3. 🎬 影片去浮水印 (時間軸修復)
- **操作方式**：
    1. 上傳影片。
    2. 點擊「截取第一幀」或等待自動加載。
    3. 在預覽圖上精確塗抹浮水印。
    4. 點擊「開始影片修復」。
- **特點**：完整保留原始音軌、幀率與畫質。
