# 更新日誌 (Changelog)

## 2026-02-13 更新
1.  **全繁體中文界面**：將所有 UI 文本翻譯為繁體中文。
2.  **新增「批量圖片處理」功能**：
    - 支持一次上傳多張圖片。
    - 在第一張圖片上設定浮水印位置後，自動應用於所有圖片。
    - 自動打包結果為 ZIP 文件供下載。
3.  **優化影片處理流程**：
    - 上傳影片後自動截取第一幀。
    - 支持在預覽圖上直觀框選浮水印區域。
    - 移除了手動輸入座標的繁瑣步驟。
4.  **修復與優化**：
    - 修復了影片處理時的預覽問題。
    - 優化了下載按鈕的體驗。
